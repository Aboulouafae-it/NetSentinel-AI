# NetSentinel AI Asset Licensing Policy

**Version:** v2.7 Planning

To ensure NetSentinel AI OS remains legally sound for commercial distribution, the following strict licensing policies apply to all visual assets (logos, icons, wallpapers, fonts, and graphics).

## 1. Allowed Assets
- **100% Original Assets:** Custom-created vectors and graphics explicitly designed for NetSentinel AI.
- **Open Source / Permissive Assets:** Assets licensed under MIT, Apache 2.0, SIL Open Font License (OFL), or CC-BY (if attribution requirements are strictly met).
- **System Fonts:** Standard distribution fonts (e.g., DejaVu, Liberation, standard X11 fonts) used via package management.

## 2. Prohibited Assets
- **NO Upstream Distribution Branding:** Absolutely no Debian swirls, Kali dragons, Ubuntu circles, or similar logos.
- **NO Vendor Logos:** Do not use hardware vendor logos (e.g., Cisco, Ubiquiti, Fortinet, AWS, Azure) in the UI or branding without explicit legal permission. Use generic icons (e.g., a generic router icon instead of a Cisco logo).
- **NO Proprietary Fonts:** Do not embed commercial fonts without purchasing the appropriate distribution license.
- **NO Unverified Icon Packs:** Do not copy icons from popular sets (FontAwesome pro, Material, etc.) unless the open-source license is verified and documented.
- **NO Unlicensed Stock Art:** No watermarked images or "found" imagery from the web.

## 3. AI-Generated Assets
- AI image generators (Midjourney, DALL-E, etc.) may be used for **ideation and concept exploration only**.
- **Commercial Use:** AI-generated images **MUST NOT** be used as final commercial assets (logos, wallpapers, UI elements) without formal legal review, manual vetting, and heavy editing/vectorization to ensure uniqueness and copyright safety.

## 4. Release Process
- **Placeholder Rule:** Placeholder assets (like SVG wireframes) must be clearly marked in their metadata or filenames as placeholders.
- **Final Review:** Any final commercial release (e.g., tagging a v1.0 ISO) requires a comprehensive asset review against this policy.
- **Source Files:** The project must retain the source files (SVG, PSD, AI) for all custom visual assets.
