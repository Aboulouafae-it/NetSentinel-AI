# NetSentinel AI Terminal Branding

This directory contains specifications and assets for the terminal and MOTD visual style.

See `motd-style.md` for the exact visual rules.

## Current Placeholder Status
The MOTD is currently a clean, text-based MOTD applied via the `includes.chroot` system. No heavy ASCII art is currently used.

## Final Asset Requirements
Any future graphical or ASCII-art additions to the MOTD or terminal prompt must pass the readability rules defined in `motd-style.md`. 

## Licensing Restrictions
No external terminal themes (e.g. Oh My Zsh specific themes) or proprietary fonts can be bundled without explicit licensing validation.
